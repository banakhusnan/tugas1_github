File ketiga ini berisi tentang pengertian Javascript (JS). JavaScript adalah bahasa pemrograman yang digunakan untuk memberikan interaksi dan fungsionalitas dinamis pada halaman web. Dengan menggunakan JavaScript, Anda dapat membuat halaman web lebih interaktif, merespons tindakan pengguna, mengubah konten secara dinamis, dan bahkan berkomunikasi dengan server untuk mengambil atau mengirim data tanpa perlu memuat ulang halaman.
